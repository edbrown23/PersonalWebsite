Verlet Integration!

This is a Javascript translation of the source code and tutorial on Verlet Integration found here. I have every intention of modifying and improving the code soon, with the ultimate goal of creating some form of skeletal animation system. However, that will likely have to wait until after graduation. In the meantime, feel free to play with the cloth simulation above.

Controls

Control Action
Shift + Left Click Creates a single new point mass
Left Click and Drag Exerts some force on the point masses near the mouse cursor in the direction of its movement
