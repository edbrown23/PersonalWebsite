Experimentation Zone!

Check out some verlet integration!

This page is an area for some experimentation with the canvas tag and javascript. Feel free to play around. The controls are described below, for reference.

Congestion on edges is conveyed by their color. A completely green edge is not congested, and a completely red edge is completely congested. The more entities on an edge, the higher the congestion.

Controls

Control Action
Left Click Create/Select a vertex
Shift Left Click Create an edge
n Spawn an entity
Space Bar Pause the simulation
